# tombot

Creating a personalised chatbot to mimic the user's speech characteristics and conversational style as closely as possible. Do you miss your friend? You can just use tombot to feel closer to never miss them again.

*This project is created with GPT-5.2 assistance - I do not claim ownership of this project

Structure

* `llm/` → model backend (Ollama, API, etc.)
* `persona/` → “how you talk”
* `memory/` → recall system (short + long term)
* `utils/` → humanization helpers
* `data/` → **never committed chat logs**
